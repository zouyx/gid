Installation
------------

如果还没有安装Go开发环境，请参考以下文档[Getting Started](http://golang.org/doc/install.html) ，安装完成后，请执行以下命令：

``` shell
gopm get github.com/zouyx/gid -v -g
```

或者

``` shell
go get -u github.com/zouyx/gid
```


*请注意*: 最好使用Go 1.8进行开发
